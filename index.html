XAML to Image

"XAML to Image" is a set of project templates for Blend for Visual Studio and/or Visual Studio. By using the templates, you can design images or icons as XAML, and export image files such as PNG, GIF, etc.

  1. Edit XAML with Blend for Visual Studio.
  2. Run the app, then image files are created.
